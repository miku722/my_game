## 🚀 一键启动（30秒开始游戏）

```bash
# 1. 克隆/创建项目
git clone https://github.com/miku722/my_game.git
cd my_game
# 或者手动复制所有文件到该目录

# 2. 安装依赖
npm install

# 3. 启动游戏
npm run dev
